# ActionCarePlus

Simple iOS app that helps users track symptoms, get on-device ML predictions for likely conditions, and see clear self-care / doctor / emergency guidance. Everything runs locally using SwiftUI, Core Data, and Core ML.

## Quick start

- **Requirements**: macOS with Xcode 16+, iOS 16+ simulator or device.
- **Open the project**: Double-click `ActionCarePlus.xcodeproj`.
- **Select the target**: Make sure the `ActionCarePlus` target is selected at the top.
- **Run**: Choose an iOS 16+ simulator (for example, iPhone 16 Pro) and press **⌘R**.

On first launch you will see onboarding, then a 4-tab app: Home, Check (symptom input), History, and Settings.
