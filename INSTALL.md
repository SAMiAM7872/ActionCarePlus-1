# Install & Run (Very Simple)

1. **Install Xcode**
   - From the Mac App Store, install **Xcode 16 or later**.

2. **Get the project**
   - Make sure the `ActionCarePlus` folder (with `ActionCarePlus.xcodeproj`) is on your Mac.

3. **Open in Xcode**
   - Double-click `ActionCarePlus.xcodeproj` to open it in Xcode.

4. **Choose a simulator**
   - In the toolbar, pick an iOS 16+ simulator (for example, *iPhone 16 Pro*).

5. **Build and run**
   - Press **⌘R** (or click the Run ▶ button).
   - The app should build, launch the simulator, and show the onboarding screens.

That’s it—no extra setup, servers, or accounts are required. All data stays on the device using Core Data and on-device ML.
